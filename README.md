### Install Gloom
#### First ``git clone``
  ``git clone https://github.com/joshDelta/Gloom-Framework.git``
#### Second ``cd``
  ``cd Gloom-Framework``
#### Third ``install``
  ``sudo python install.py``
#### Run The Program
  ``sudo python gloom.py``
